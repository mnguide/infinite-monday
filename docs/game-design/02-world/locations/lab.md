| 항목 | 내용 |

## 상세 설명

| 조건 | 방법 |
|------|------|

| 침입 | 야간 비상계단 (루프 10+, 위험) |

### 분위기 묘사

> 12층의 문이 열린다.
> 차가운 공기와 함께 서버 냉각 소리가 들린다.
> 하얀 조명 아래, 수십 개의 모니터가 푸른 빛을 내뿜는다.
>
> 여기가... "Project Monday"의 본거지인가.

## 게임플레이 역할

### 핵심 미스터리 장소

- 모든 의문의 답이 있는 곳
- 루프의 원인 공개
- 플레이어 정체성 폭로

### 엔딩 분기

- 탈출 엔딩의 필수 경로
- 비밀 엔딩 루트
- 진실 대면

### 긴장감 최고조

- 접근 자체가 어려움
- 들키면 큰 페널티
- 시간 압박

## 연출 노트

### 시각적 연출

```
색감: 차가운 파랑/하얀, 무균실 느낌
조명: 형광등, 모니터 불빛
질감: 금속, 유리, 차가움
```

### 사운드 연출 (텍스트)

```
"윙..." - 서버 냉각팬
"삐..." - 전자 기기 소리
"딸깍" - 키보드
정적 - 인기척 없음
```

### 분위기 키워드

```
비밀, 진실, 충격
차가움, 무균, 실험실
공포, 긴장, 발각 위험
```
