# 7층 사무실 (개발1팀)

## 기본 정보

| 항목 | 내용 |
|------|------|
| ID | `office` |
| 위치 | 넥스트엣지 빌딩 7층 |
| 운영 시간 | 24시간 (야간 출입 가능) |
| 분위기 | 전형적인 IT회사 사무실, 모니터 빛 |

### 분위기 묘사

> 개발1팀 사무실이다.
> 5개의 책상이 창가를 향해 나란히 놓여 있고,
> 각자의 모니터에서 푸른 불빛이 새어 나온다.
> 키보드 두드리는 소리와 에어컨 소리가 배경음처럼 흐른다.

### 시간대별 변화

| 시간 | 상태 | 분위기 |
|------|------|--------|
| 09:00~12:00 | 전원 출근, 회의 | 활기참 |
| 12:00~13:00 | 점심시간, 비어있음 | 조용함 |
| 14:00~18:00 | 업무 집중 | 나른함 |
| 18:00~20:00 | 일부 퇴근 | 한산함 |
| 20:00~22:00 | 야근조만 | 긴장감 |
| 22:00~ | 거의 무인 | 으스스함 |

- **김부장**: 14:30~16:30 자리 비움, 17:00~19:30 또 자리 비움
- **이승호**: 13:30 옥상 흡연
- **준혁**: 야근 잦음, 혼자 남아있을 때 말 많아짐
- **민수**: 가장 먼저 퇴근, 잡무 담당

  - text: "집중해서 듣는다"

## 게임플레이 역할

### 메인 허브

- 게임의 중심 공간
- 대부분의 업무 시간 이곳에서 진행
- 동료들과의 기본 상호작용

### 정보 수집

- 부장 책상 = 핵심 단서
- 동료 관찰 = 행동 패턴 파악
- 야근 = 숨겨진 정보 접근

### 관계 발전

- 업무 협력 = 신뢰 쌓기
- 개인 대화 = 호감도 상승
- 비밀 공유 = 깊은 관계

## 연출 노트

### 분위기 전환

```
오전: 활기차고 평범한 직장
오후: 나른하고 시간이 안 가는 느낌
저녁: 긴장감, 무언가 숨겨진 듯
심야: 미스터리, 공포에 가까운 분위기
```

### 소리 연출 (텍스트로)

```
- 키보드 타닥타닥
- 마우스 클릭
- 에어컨 웅웅
- 전화 벨소리 (가끔)
- 프린터 위이잉 (가끔)
```
