## 주요 장소 (상세)

### 7층 - 개발1팀 사무실

**구조**
```
┌─────────────────────────────────────┐
│  [창문]                              │
├─────────────────────────────────────┤
│                                      │
│  ┌────┐  ┌────┐  ┌────┐  ┌────┐    │
│  │부장 │  │과장 │  │대리 │  │사원 │    │ 좌석 배치
│  │김동현│  │이승호│  │(나)│  │준혁 │    │
│  └────┘  └────┘  └────┘  └────┘    │
│                                      │
│  ┌────┐                              │
│  │인턴 │        [회의 테이블]          │
│  │민수 │                              │
│  └────┘                              │
│                                      │

│  [출입문]              [프린터/복합기] │

```

**발견 가능한 단서**
- 김부장 책상 서류 (특정 조건)
- 휴지통의 메모 조각
- 부장 PC 화면 (몰래 볼 때)

### 4층 - 마케팅팀 사무실

**특징**
- 윤지현 소속
- 개발팀보다 화사한 분위기
- 플레이어가 갈 구실이 필요함

**이벤트**
- 지현과의 대화
- 마케팅 자료 요청 핑계
- 우연한 마주침

### 12층 - 연구소 (비밀 구역)

**특징**
- 일반 직원 출입 금지
- 카드키 필요
- Project Monday의 본거지

**진입 조건**
- 김부장의 카드키 획득 (루프 10+)
- 또는 야간 침입 이벤트

**발견 가능한 단서**
- 프로젝트 문서
- 실험 로그
- 플레이어 관련 파일

### 옥상

**특징**
- 휴식/흡연 장소
- 깊은 대화의 장소
- 비밀 이야기가 오가는 곳

**이벤트**
- 김부장과의 대화 (호감도 높을 때)
- 지현과의 고백
- 준혁의 비밀

### 1층 - 카페 "빈틈"

**특징**
- 사내 카페
- 휴식 + 만남의 장소
- 아침/점심/저녁 시간대별 다른 분위기

**이벤트**
- 우연한 만남
- 커피 심부름
- 소문 듣기

### B1층 - 구내식당

**특징**
- 점심시간 북적임
- 같이 먹는 사람 = 관계 발전
- 대화 엿듣기 가능

**이벤트**
- 팀 점심
- 다른 팀 동료와 식사
- 혼밥

- 평범한 IT 회사
- 새 프로젝트 개발 중

- 직원들이 피실험자

- 플레이어의 과거와 연결
