## 핵심 분기점

플레이어의 선택에 따라 달라지는 주요 시점들.

### 분기점 1: 아침 (08:00~09:00)

**출근 방법 선택**
```
┌─ 일찍 출근 (08:00)
│    └→ 카페에서 지현 마주침
│
├─ 정시 출근 (08:45)
│    └→ 평범한 하루 시작
│
└─ 지각 (09:15)
     └→ 부장에게 혼남, 평판 하락
```

### 분기점 2: 아침 회의 (09:00)

**회의 중 태도**
```
┌─ 적극 참여
│    └→ 부장 호감도 +, 업무 증가
│
├─ 조용히 있기
│    └→ 변화 없음
│
└─ 딴생각 (회의 내용 놓침)
     └→ 이후 모르는 정보 발생
```

### 분기점 3: 점심 (12:00~13:00)

**점심 파트너 선택**
```
┌─ 팀과 함께
│    └→ 팀 정보 획득, 일반 대화
│
├─ 지현과 단둘이
│    └→ 지현 호감도 ++, 로맨스 진행
│
├─ 준혁과 단둘이
│    └→ 준혁 호감도 ++, 비밀 힌트
│
├─ 혼밥
│    └→ 시간 절약, 다른 이벤트 참여 가능
│
└─ 식당 탐문
     └→ 다른 부서 정보 획득
```

### 분기점 4: 오후 자유시간 (14:00~17:00)

**김부장 자리 비움 동안**
```
┌─ 업무 계속
│    └→ 평범한 진행
│
├─ 부장 책상 조사
│    └→ 단서 획득 (위험)
│
├─ 다른 층 탐색
│    └→ 새로운 정보
│
└─ 옥상 휴식
     └→ 승호와 대화 기회
```

### 분기점 5: 김부장 12층행 (17:00)

**미행 여부**
```
┌─ 미행한다
│    ├→ 성공: 12층 정보
│    └→ 실패: 발각, 평판 대폭 하락
│
└─ 미행 안 함
     └→ 평범한 진행
```

### 분기점 6: 퇴근 (18:00~)

**퇴근 시점 선택**
```
┌─ 정시 퇴근 (18:00)
│    ├→ 지현과 같이 퇴근 기회
│    └→ 개인 시간 확보
│
├─ 약간 야근 (20:00)
│    ├→ 부장과 대화 기회
│    └→ 멘탈 소모
│
└─ 심야 야근 (22:00+)
     ├→ 12층 탐색 기회
     └→ 체력/멘탈 대폭 소모
```

## 이벤트 충돌 처리

같은 시간에 여러 이벤트가 가능할 때:

### 우선순위

1. 강제 이벤트 (스토리 필수)
2. 플레이어 선택 이벤트
3. 캐릭터 연동 이벤트
4. 랜덤 이벤트

### 예시

**12:00 점심시간**
- 팀 점심 (기본)
- 지현과 점심 (선택)
- 준혁과 점심 (선택)
- 혼밥 (선택)
- 갑작스러운 회의 (랜덤 이벤트)

→ 랜덤 이벤트 발생 시, 선택한 점심 이벤트 취소
→ 또는 회의 후 늦은 점심으로 진행
