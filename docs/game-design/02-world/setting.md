|------|------|

| 계절 | 초여름 (6월) |

| 날씨 | 맑음 (기본), 비 (랜덤) |

### 타임루프 규칙

#### 루프 시작
- 매일 오전 7시, 플레이어 자택 침대에서 시작
- 알람 소리와 함께 눈을 뜸
- 이전 루프의 물리적 상태는 리셋
- 기억은 유지됨

#### 루프 종료 조건
1. **자연 종료**: 자정(00:00) 도달
2. **기절**: 체력 0 도달
3. **정신 붕괴**: 멘탈 0 도달
4. **강제 종료**: 특정 이벤트

#### 루프 간 유지되는 것
- 플레이어의 기억 (단서, 정보)
- 캐릭터에 대한 인상 (호감도)
- 메타 지식 (선택지 해금)
- 달성한 엔딩 기록

#### 루프 간 리셋되는 것
- 신체 상태 (체력, 부상)
- 소지품 (물건, 돈)
- 다른 캐릭터들의 기억
- 그날 일어난 사건들

### 루프의 원인 (스포일러)

> **주의**: 이 섹션은 메인 스토리의 핵심 반전을 포함합니다.

**표면적 원인**: 불명

**진짜 원인 (점진적 공개)**:
1. 처음엔 초자연적 현상으로 보임
2. 회사의 비밀 프로젝트와 연관
3. 플레이어 본인의 선택이 관련

**최종 진실**:
- "Project Monday" - 업무 효율 극대화 프로젝트
- 직원들의 의식을 시뮬레이션에 가둠
- 완벽한 하루를 찾을 때까지 반복
- 플레이어는 실험 대상자 중 유일한 각성자

## 사회적 배경

### 대한민국 직장 문화 (게임 내)

| 요소 | 게임 내 반영 |
|------|--------------|
| 야근 문화 | 퇴근 시간 = 선택, 야근 = 스탯 변화 |
| 회식 | 이벤트로 등장, 관계도 변화 |
| 상하관계 | 부장/과장/대리/사원 호칭 시스템 |
| 점심 문화 | 동료와 점심 = 호감도 이벤트 |
| 카페 문화 | 카페 = 휴식/대화 장소 |

### 시대상

- 스마트폰/메신저 일상화
- 재택근무 언급 (하지만 플레이어는 출근)
- 워라밸 담론
- MZ세대 vs 기성세대 갈등

### 설정 구멍 대응

**Q: 왜 플레이어만 기억하나요?**
A: 메인 미스터리의 핵심. 점진적 공개.

**Q: 왜 도망가지 않나요?**
A: 1) 첫 루프들은 현실인 줄 앎 2) 나중엔 탈출하려 함

**Q: 다른 사람도 루프 중인가요?**
A: 스포일러. 특정 조건에서 밝혀짐.

**Q: 왜 월요일인가요?**
A: 1) 상징성 (싫은 날) 2) 스토리 이유 있음
