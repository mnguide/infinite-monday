# 에피소드 1: 끝없는 월요일

## 에피소드 개요

| 항목 | 내용 |
|------|------|
| 부제 | 끝없는 월요일 (Endless Monday) |
| 루프 범위 | 1 ~ 무한 (엔딩까지) |
| 예상 플레이 | 2~3시간 (메인 엔딩) |
| 엔딩 수 | 10개 |

## 1막: 발단 (루프 1~3)

### 주요 이벤트

| 루프 | 핵심 이벤트 | 목적 |
|------|-------------|------|
| 1 | 평범한 월요일 | 일상 체험, 시스템 학습 |
| 2 | "어? 또 월요일?" | 반복 인식, 혼란 |
| 3 | 확신과 적응 | 루프 확정, 탐험 시작 |

### 루프 1: 평범한 하루

```yaml
scenes:
  - scene_home_morning (자택)
  - scene_commute (출근)
  - scene_office_morning_meeting (아침회의)
  - scene_office_work (업무)
  - scene_cafeteria_team_lunch (점심)
  - scene_office_work_afternoon (오후)
  - scene_leave_office (퇴근)
  - scene_home_return (귀가)

purpose:
  - 게임 조작 학습
  - 캐릭터 소개
  - 기본 루틴 파악

ending: 자연스러운 하루 종료 → 다음 날 아침
```

### 루프 2: 데자뷰

```yaml
scenes:
  - scene_home_morning_2 (이상한 기시감)
  - scene_commute (익숙한 출근길)
  - scene_office_morning_meeting_2 (똑같은 회의)

new_dialogue:
  player: "...뭔가 이상해. 이거 어제도..."
  player: "아니, 그럴 리가."

purpose:
  - 반복 인식 시작
  - 플레이어 혼란
  - 의문 제기
```

### 루프 3: 확신

```yaml
scenes:
  - scene_home_morning_3 (또 월요일)
  - scene_commute_dejavu (확신)

new_dialogue:
  player: "...이건 꿈이 아니야."
  player: "진짜로 월요일이 반복되고 있어."
  player: "왜? 어떻게?"

unlocks:
  - 자유 탐험 모드
  - 새로운 선택지들
  - 단서 수집 시작
```

## 2막: 전개 (루프 4~7)

### 주요 이벤트

| 루프 | 핵심 이벤트 | 해금 요소 |
|------|-------------|-----------|
| 4 | 자유 탐험 시작 | 장소 이동 자유 |
| 5 | 이상 징후 발견 | 12층 언급 |
| 6 | 관계 심화 | 호감도 이벤트 |
| 7 | 부장 미행 | 12층 접근 시도 |

### 핵심 씬

```yaml
scene_cafe_rumor:
  trigger: 루프 5+, 카페 방문
  content: 바리스타가 12층 소문 언급
  clue: clue_12f_rumor

scene_rooftop_boss_call:
  trigger: 루프 6+, 옥상, 부장 추적
  content: 부장의 의심스러운 통화
  clue: clue_boss_call

scene_jihyun_deep_talk:
  trigger: 루프 5+, 지현 호감도 40+
  content: 지현과의 진솔한 대화
  relationship: 로맨스 플래그
```

## 3막: 위기 (루프 8~12)

### 주요 이벤트

| 루프 | 핵심 이벤트 | 결과 |
|------|-------------|------|
| 8-9 | 12층 첫 접근 시도 | 실패, 정보 수집 |
| 10 | 준혁의 진실 공개 | 충격, 동지 발견 |
| 11-12 | 탈출 계획 수립 | 루트 분기 |

### 준혁 공개 씬

```yaml
scene_rooftop_junhyuk_truth:
  trigger: 루프 10+, 야간 옥상, 준혁 호감도 60+
  content: |
    "선배, 저 말이에요."
    "지금 몇 번째 월요일인지 아세요?"
    "저는요... 100번째입니다."

  impact:
    - 스토리 전환점
    - 협력 루트 개방
    - 새로운 정보 획득
```
