## 메인 엔딩

### ESCAPE (탈출)

**조건:**
```yaml
conditions:
  loops: >= 7
  stats:
    health: >= 50
    mental: >= 50
    reputation: >= 30
  flags:
    lab_discovered: true
    system_shutdown: true
  relationship: (any >= 60) # 누구든 깊은 관계
```

**연출:**
```
[장소: 자택]

눈을 떴다.

...알람이 울리지 않는다.
아니, 울리는데... 소리가 다르다.

핸드폰을 확인한다.

"6월 8일 화요일"

...화요일.

루프가 끝났다.
나는 해냈다.
```

**후일담:**
```
회사에 출근한다.
모든 게 달라 보인다.
하지만 어제의 기억은 생생하다.

김부장과 눈이 마주친다.
그가 미세하게 고개를 끄덕인다.

...그도 기억하는 걸까?

어쨌든, 오늘은 화요일이다.
새로운 하루의 시작이다.
```

### SECRET (비밀)

**조건:**
```yaml
conditions:
  loops: >= 15
  flags:
    all_clues: true
    junhyuk_cooperation: true
    boss_persuaded: true
    full_truth: true
  hidden:
    discovered_origin: true  # 플레이어 과거와의 연결
```

**연출:**
```
[장소: 12층 연구소 서버실]

모든 진실을 알게 됐다.

Project Monday. 의식 시뮬레이션.
그리고... 내가 이 프로젝트에 참여한 이유.

화면에 뜬 파일을 본다.

"지원서 - [플레이어 이름]"
"동의 일자: 20XX년 X월 X일"
"동의 사유: [편집됨]"

...내가 자원한 거였어?

기억이 돌아온다.
1년 전. 그날.
모든 것을 잊고 싶었던 그날.

나는 스스로 이곳에 들어왔다.
```

**선택지:**
```yaml
choices:
  - text: "모든 걸 공개한다"
    next: secret_ending_expose

  - text: "프로젝트를 이어간다"
    next: secret_ending_continue

  - text: "기억을 지우고 떠난다"
    next: secret_ending_forget
```
