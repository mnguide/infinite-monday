# 메인 미스터리

## 핵심 질문

게임 전체를 관통하는 미스터리:

> **"왜 월요일이 반복되는가?"**

이 질문에 대한 답을 찾아가는 것이 게임의 핵심.

---

## 진실의 레이어

### Layer 0: 표면 (루프 1~3)

**플레이어가 아는 것:**
- 월요일이 반복된다
- 나만 기억을 유지한다
- 다른 사람들은 모른다

**의문:**
- 왜 나만?
- 언제까지?
- 어떻게 벗어나지?

### Layer 1: 의심 (루프 4~7)

**새로 알게 되는 것:**
- 12층에 비밀 연구소가 있다
- 김부장이 수상하게 행동한다
- 회사에 뭔가 숨겨져 있다

**단서:**
- 부장의 12층 출입
- 옥상에서의 통화
- 바리스타의 소문

### Layer 2: 접근 (루프 8~12)

**새로 알게 되는 것:**
- "Project Monday"의 존재
- 준혁도 루프를 인식한다 (100번째)
- 직원들이 실험 대상이다

**단서:**
- 준혁의 고백
- 12층 탐색
- 연구 문서

### Layer 3: 진실 (루프 13+)

**최종 진실:**
- Project Monday = 의식 시뮬레이션 실험
- 목적: "완벽한 업무 효율의 하루" 찾기
- 플레이어는 처음부터 대상자
- 탈출 조건: 시스템이 정의한 "완벽한 하루" 달성

## Project Monday 상세

### 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 정식 명칭 | Project Monday: 업무 효율 극대화 연구 |
| 시작 | 5년 전 |
| 주관 | 넥스트엣지 연구소 |
| 목적 | 직원 의식 시뮬레이션을 통한 최적 업무 패턴 발견 |

### 프로젝트 메커니즘

```
1. 대상자 선발
   └→ 입사자 중 "적합 인원" 스크리닝

2. 시뮬레이션 투입
   └→ 본인 인지 없이 루프에 진입

3. 하루 시뮬레이션
   └→ 월요일 하루를 반복 체험

4. 조건 체크
   └→ "완벽한 하루" 달성 여부 평가

5-A. 미달성
   └→ 기억 리셋, 루프 반복

5-B. 달성
   └→ 루프 종료, 현실 복귀
```

### "완벽한 하루" 정의

시스템이 정의한 탈출 조건:

```yaml
perfect_day:
  stats:
    health: >= 80
    mental: >= 80
    reputation: >= 70

  achievements:
    - 업무 완수
    - 갈등 해결
    - 관계 형성 (호감도 50+ 1인 이상)
    - 정시 퇴근 또는 의미 있는 야근

  secret_condition:
    - 루프 인식 상태에서 "선택" 수행
    - = 자유 의지의 증명
```

### 각성 현상

일부 대상자에게서 나타나는 예외:

- 루프 기억 유지
- 원인 불명 (시스템 버그? 의도?)
- 준혁: 첫 번째 각성자 (100+ 루프)
- 플레이어: 두 번째 각성자

## 핵심 캐릭터별 비밀

### 김동현 (부장)

```yaml
surface: 엄격한 팀장
secret:
  - Project Monday 현장 책임자
  - 5년간 실험 관리
  - 내부 갈등 중 (죄책감)
  - 탈출구 찾는 중일지도
```

### 박준혁 (사원)

```yaml
surface: 조용한 신입
secret:
  - 첫 번째 각성자
  - 100번째 루프 진행 중
  - 50번째 루프에서 포기
  - 플레이어에게서 희망 발견
```

### 플레이어

```yaml
surface: 평범한 대리
secret:
  - 입사 시 대상자로 선발
  - 각성 잠재력 높음
  - 프로젝트의 핵심 변수
  - 과거의 선택이 현재와 연결 (?)
```

## 엔딩별 진실 공개

### ESCAPE 엔딩

```
공개 범위: 탈출 방법
숨겨진 것: 프로젝트 전모
플레이어 상태: 탈출 성공, 일부 진실 인지
```

### SECRET 엔딩

```
공개 범위: 프로젝트 전모 + 배후
숨겨진 것: 없음 (완전 공개)
플레이어 상태: 모든 진실 파악, 선택의 갈림길
```

### 기타 엔딩

```
PROMOTION: 루프 내 성공, 탈출 실패
LOVE: 관계 형성, 함께 탈출
STARTUP: 현실 도피형 탈출
BURNOUT: 포기
```

## 플레이어 인지 관리

### 알려주는 것

- 루프 사실 (즉시)
- 회사에 비밀 있음 (루프 5+)
- 연구소 존재 (루프 7+)
- 동료 루퍼 존재 (루프 10+)

### 숨기는 것 (스스로 발견)

- 왜 자신이 각성했는지
- 탈출 정확한 조건
- 프로젝트 배후

### 절대 직접 알려주지 않는 것

- 시스템 작동 원리 전체
- 다른 대상자 명단
- 프로젝트 종료 후 계획
