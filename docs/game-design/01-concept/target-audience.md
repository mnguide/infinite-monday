# 타겟 오디언스

## 핵심 타겟

### 프라이머리: 직장인 게이머 (25-35세)

**데모그래픽**
- 나이: 25~35세
- 직업: 사무직 직장인
- 성별: 성별 무관 (약간 남성 편향 예상)
- 거주: 수도권 대도시

**사이코그래픽**
- 학창시절 게임을 즐겼으나 시간 부족
- 출퇴근 시간 활용하고 싶음
- 복잡한 게임보다 쉽게 접근 가능한 것 선호
- 스토리/세계관에 몰입하는 것을 좋아함

**행동 특성**
- 토스 앱 일 1회 이상 사용
- 모바일 게임 세션: 5~15분
- 게임 결제 경험 있으나 소액 선호
- SNS에서 게임 밈/리뷰 공유

**페인 포인트**
- "월요일 출근하기 싫다"
- "시간이 없어서 게임을 못 해"
- "요즘 게임은 너무 복잡해"
- "광고가 너무 많은 게임은 싫어"

### 세컨더리: 미스터리/추리 게임 팬

**데모그래픽**
- 나이: 20~40세
- 성별: 성별 무관
- 플랫폼: PC/콘솔/모바일 다양

**사이코그래픽**
- 역전재판, 단간론파 등 추리 게임 경험
- 드라마/영화에서 반전을 좋아함
- 복선 회수에 희열을 느낌
- 커뮤니티에서 공략/추리 토론 참여

**행동 특성**
- 스토리 완결까지 플레이
- 히든 엔딩 수집 욕구 강함
- 게임 스크린샷/짤 공유
- 정주행 플레이 패턴

### 터셔리: 타임루프 콘텐츠 팬

**데모그래픽**
- 나이: 18~45세
- 특징: SF/판타지 장르 팬

**관련 콘텐츠 경험**
- 영화: 엣지 오브 투모로우, 해피 데스 데이, 소스 코드
- 드라마: 러시안 돌, 낮과 밤
- 게임: Outer Wilds, 12 Minutes, Deathloop

## 비타겟 (Non-Target)

### 하드코어 게이머
- 고사양 AAA 게임 선호
- 모바일 게임을 "진짜 게임"으로 인정 안 함
- **이유**: 텍스트 기반 게임에 만족 못 함

### 캐주얼 퍼즐 게이머
- 매치-3, 단순 퍼즐 선호
- 스토리보다 즉각적 보상 중시
- **이유**: 스토리 읽기를 귀찮아함

### 10대
- 직장 생활 공감 불가
- 텍스트 게임 경험 부족
- **이유**: 핵심 설정에 공감 어려움

### 게임 비이용자
- 게임을 전혀 하지 않음
- 스마트폰 = 실용 목적만
- **이유**: 전환 비용 너무 높음

## 유저 페르소나

### 페르소나 1: 김대리 (30세, 남성)

**프로필**
- 대기업 마케팅팀 5년차
- 매일 지하철 출퇴근 (왕복 2시간)
- 주중 저녁 약속 1~2회

**게임 이력**
- 어릴 때 역전재판 시리즈 올클리어
- 요즘은 출퇴근 때 가챠 게임 데일리만
- "예전처럼 스토리 게임 하고 싶다"

**플레이 패턴**
- 출근길 20분 + 퇴근길 20분
- 점심시간 가끔 10분
- 주말에 몰아서 1시간

**기대하는 것**
- 짧은 시간에 진전 느끼기
- 추리하는 재미
- "아 이거였어!" 하는 순간
- 광고는 적게, 있어도 선택적으로

**우려하는 것**
- 진행이 막히면 답답
- 너무 어려우면 포기
- 시간 낭비 같은 느낌

### 페르소나 2: 이주임 (27세, 여성)

**프로필**
- 스타트업 기획자 3년차
- 재택근무 주 2회
- 로맨스 소설/웹툰 구독

**게임 이력**
- 여성향 게임 (연애 시뮬레이션) 경험
- 스토리 게임 좋아하지만 찾기 어려움
- 유튜브로 게임 스트리밍 시청

**플레이 패턴**
- 퇴근 후 집에서 30분~1시간
- 침대에서 잠들기 전
- 관심 가면 주말 정주행

**기대하는 것**
- 매력적인 캐릭터들
- 로맨스 요소
- 감정적 몰입
- 여러 엔딩 수집

**우려하는 것**
- 캐릭터가 매력 없으면 흥미 없음
- 선택이 의미 없으면 허탈함
- 불쾌한 내용 (과도한 폭력/선정성)

### 페르소나 3: 박과장 (38세, 남성)

**프로필**
- 중견기업 개발팀 팀장
- 육아 중 (초등학생 자녀)
- 게임 할 시간 거의 없음

**게임 이력**
- 90년대 어드벤처 게임의 추억
- 요즘은 연 1~2개 게임만
- "시간이 없어서" 게임 포기

**플레이 패턴**
- 극히 불규칙 (일주일에 1~2회)
- 한 번에 5~10분
- 자투리 시간 활용

**기대하는 것**
- 언제 그만둬도 괜찮은 구조
- 복귀 시 이전 내용 상기
- 복잡하지 않은 조작
- "내 과거 같다" 공감

**우려하는 것**
- 진행을 까먹을 것 같음
- 나이 들어서 반응속도 게임은 어려움
- 애들한테 들키면 부끄러움 (?)

## 시장 규모 추정

### TAM (Total Addressable Market)

**한국 모바일 게임 시장**
- 모바일 게임 이용자: 약 2,500만 명
- 월 1회 이상 플레이: 약 1,800만 명

### SAM (Serviceable Available Market)

**토스 앱 게임 이용자**
- 토스 앱 MAU: 약 1,500만 명
- 토스 게임 이용 비율: 약 15%
- = 약 225만 명

### SOM (Serviceable Obtainable Market)

**목표 획득 유저**
- 스토리/추리 게임 선호: SAM의 10%
- = 약 22만 명

**현실적 1년차 목표**
- DAU: 1~2만 명
- MAU: 5~8만 명
- 설치 수: 10~15만

## 유저 저니 맵

### 인지 단계 (Awareness)

**터치포인트**
- 토스 앱 내 게임 탭
- 지인 추천/공유
- 게임 커뮤니티 입소문

**유저 생각**
- "토스에 게임이 있네?"
- "무한 월요일? 뭔가 공감되는 제목"
- "무료라니까 해볼까"

### 시작 단계 (Onboarding)

**터치포인트**
- 게임 첫 실행
- 튜토리얼 (루프 1)

**유저 생각**
- "어 월요일이 반복된다고?"
- "직장인 설정이네, 현실적이다"
- "선택지가 있구나"

**목표**
- 5분 내 첫 루프 경험
- 기본 조작 이해
- 호기심 유발

### 탐색 단계 (Exploration)

**터치포인트**
- 루프 2~5
- 다양한 선택지 시도
- 첫 번째 엔딩 도달

**유저 생각**
- "아 이걸 이렇게도 할 수 있구나"
- "이 캐릭터 뭔가 숨기고 있어"
- "엔딩이 여러 개인 것 같은데?"

**목표**
- 시스템 완전 이해
- 단서 수집의 재미
- 리텐션 확보

### 몰입 단계 (Engagement)

**터치포인트**
- 루프 6~12
- 메인 스토리 진행
- 엔딩 수집

**유저 생각**
- "다음이 궁금해서 멈출 수가 없어"
- "아 그래서 그랬구나!"
- "히든 엔딩이 있대, 찾아야지"

**목표**
- 정기적 플레이 습관
- 수익화 포인트 도달
- 커뮤니티 참여

### 완료 단계 (Completion)

**터치포인트**
- 메인 엔딩 도달
- 도감 완성

**유저 생각**
- "다 깼다! 뿌듯해"
- "다른 사람한테 추천해야지"
- "시즌 2 나오면 또 할 거야"

**목표**
- 긍정적 완료 경험
- 바이럴 유도
- 확장 콘텐츠 기대

### 이탈 방지 포인트

| 단계 | 이탈 위험 | 방지책 |
|------|-----------|--------|
| 시작 | "재미없어 보여" | 첫 5분 임팩트 |
| 탐색 | "똑같은 거 반복" | 새 단서/이벤트 |
| 몰입 | "막혀서 못 나가" | 힌트 시스템 |
| 완료 | "할 게 없어" | 히든 콘텐츠 |
