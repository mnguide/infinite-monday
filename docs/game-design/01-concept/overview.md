# 게임 개요

## 컨셉

**"월요일이 반복된다. 왜? 회사에 숨겨진 비밀 때문이다."**

평범한 직장인인 플레이어는 어느 날 월요일이 무한 반복되고 있음을 깨닫는다.
처음엔 혼란스럽지만, 곧 이 루프를 이용해 회사에 숨겨진 비밀을 파헤치기 시작한다.
단서를 모으고, 동료들의 진실을 알아가며, 결국 루프의 원인을 밝혀 탈출해야 한다.

## 핵심 경험

### 1. 추리의 쾌감
- 흩어진 단서들을 연결
- 모순된 증언 발견
- "아하!" 하는 깨달음의 순간

### 2. 타임루프의 재미
- 실패해도 괜찮다 (다시 하면 됨)
- 전 루프의 정보가 다음 루프에서 무기가 됨
- 같은 하루를 다르게 살기

### 3. 공감과 몰입
- 누구나 겪어본 월요일 아침
- 현실적인 직장 생활
- 감정이입 되는 캐릭터들

### 4. 전략적 선택
- 제한된 시간을 어디에 쓸 것인가
- 누구를 먼저 조사할 것인가
- 어떤 증거를 제시할 것인가

## 레퍼런스

| 게임 | 참고 요소 |
|------|-----------|
| 역전재판 | 증거 제시, 모순 찾기, 심문 |
| Outer Wilds | 타임루프, 정보 수집, 탐험 |
| Slay the Spire | 전략적 선택, 리소스 관리 |
| 페르소나 시리즈 | 시간 관리, 관계 시스템 |

## 차별화 포인트

1. **직장인 설정** - 판타지/SF가 아닌 현실 공감
2. **미스터리 + 타임루프** - 두 장르의 시너지
3. **모바일 최적화** - 짧은 세션으로 플레이 가능
4. **한국 정서** - 한국 직장 문화 반영

## 목표 플레이 시간

- **1회 플레이 (메인 엔딩)**: 2~3시간
- **완전 클리어 (모든 엔딩 + 도감)**: 8~10시간
- **1루프 (한 회차)**: 15~20분

## 성공 지표

### 정성적
- "다음이 궁금해서 멈출 수가 없다"
- "아 이거였어?! 소름"
- "한 번 더 해봐야지"

### 정량적
- 1일 리텐션: 40% 이상
- 7일 리텐션: 20% 이상
- 평균 세션: 10분 이상
- 엔딩 도달률: 30% 이상
