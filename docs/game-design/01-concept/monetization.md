# 수익 모델

## 기본 원칙

### 1. Free-to-Play Friendly
- 모든 엔딩 무과금으로 달성 가능
- 광고는 선택적 (강제 광고 없음)
- 유료 요소는 "편의" 또는 "추가 콘텐츠"

### 2. 플레이어 존중
- 진행 막는 페이월 없음
- 광고 시청 = 플레이어의 명시적 선택
- 가치 있는 보상만 제공

### 3. 수익 지속성
- 1회성 구매보다 반복 시청 유도
- 콘텐츠 확장으로 장기 수익

## 수익원 1: 보상형 광고 (Rewarded Ads)

### 광고 포인트

#### 1. 시간 연장 (Time Extension)
**상황**: 하루가 끝나려 할 때 (21:00 이후)
**제안**: "조금 더 할 수 있어... 광고를 보시겠습니까?"
**보상**: +2시간 (23:00까지 연장)
**제한**: 루프당 1회만 가능

```
┌─────────────────────────────────────┐
│  벌써 9시야...                        │
│  아직 하고 싶은 게 있는데.             │
│                                      │
│  ┌───────────────────────────────┐  │
│  │  🎬 광고를 보고 시간 연장하기   │  │
│  │     (+2시간)                   │  │
│  └───────────────────────────────┘  │
│                                      │
│  ┌───────────────────────────────┐  │
│  │  💤 그냥 집에 가기              │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

**예상 전환율**: 20~30%
**일 시청 횟수**: DAU × 0.5회

#### 2. 힌트 보기 (Hint System)
**상황**: 플레이어가 막혔을 때 (같은 씬 3회 이상 방문)
**제안**: "힌트가 필요하신가요?"
**보상**: 현재 목표 관련 힌트 제공

**힌트 레벨**:
| 레벨 | 내용 | 비용 |
|------|------|------|
| 1 | 방향 힌트 | 무료 (1일 1회) |
| 2 | 구체적 힌트 | 광고 1회 |
| 3 | 정답 | 광고 2회 |

```
┌─────────────────────────────────────┐
│  💡 힌트 시스템                       │
├─────────────────────────────────────┤
│  Q: 김부장의 비밀을 알고 싶다          │
│                                      │
│  힌트 1: "옥상에서 뭔가 단서가..."     │
│          [무료 - 오늘 사용함]          │
│                                      │
│  힌트 2: [🎬 광고 보고 확인하기]       │
│                                      │
│  힌트 3: [🎬🎬 광고 2회 보고 정답]     │
└─────────────────────────────────────┘
```

**예상 전환율**: 10~15%
**일 시청 횟수**: DAU × 0.2회

#### 3. 보너스 머니 (Bonus Money)
**상황**: 루프 시작 시 랜덤 제안
**제안**: "오늘 용돈이 생길 것 같은 예감..."
**보상**: +50,000원 (기본 시작금의 2배)
**빈도**: 3루프에 1회 정도 제안

```
┌─────────────────────────────────────┐
│  🍀 오늘은 뭔가 운이 좋은 것 같다...   │
│                                      │
│  ┌───────────────────────────────┐  │
│  │  🎬 광고를 보고 보너스 받기     │  │
│  │     (+50,000원)               │  │
│  └───────────────────────────────┘  │
│                                      │
│  ┌───────────────────────────────┐  │
│  │  😌 됐어, 그냥 시작할게         │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

**예상 전환율**: 30~40%
**일 시청 횟수**: DAU × 0.3회

#### 4. 리와인드 (Rewind)
**상황**: 나쁜 선택 후 후회할 때
**제안**: "방금 선택을 되돌리시겠습니까?"
**보상**: 직전 씬으로 복귀
**제한**: 루프당 2회까지

```
┌─────────────────────────────────────┐
│  ❌ 실수한 것 같다...                 │
│                                      │
│  ┌───────────────────────────────┐  │
│  │  🎬 광고를 보고 되돌리기        │  │
│  │     (직전 선택으로)            │  │
│  └───────────────────────────────┘  │
│                                      │
│  ┌───────────────────────────────┐  │
│  │  🤷 괜찮아, 다음 루프에...       │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

**예상 전환율**: 15~25%
**일 시청 횟수**: DAU × 0.2회

### 광고 수익 예상

| 지표 | 값 |
|------|-----|
| DAU | 10,000명 (목표) |
| 일 평균 광고 시청 | 1.2회/유저 |
| 총 일 시청 | 12,000회 |
| CPM (한국) | $3~5 |
| 일 수익 | $36~60 |
| 월 수익 | $1,080~1,800 |

## 수익원 2: 인앱 구매 (IAP)

### 구매 아이템

#### 1. 프리미엄 패스 (Premium Pass)
**가격**: ₩4,900 (일회성)
**내용**:
- 광고 제거 (모든 보상 무료 획득)
- 힌트 무제한
- 시간 연장 무제한
- 리와인드 무제한
- 전용 UI 테마

**타겟**: 광고 싫어하는 유저, 열성 팬

#### 2. 힌트 패키지
| 패키지 | 힌트 수 | 가격 |
|--------|---------|------|
| 소형 | 5회 | ₩1,000 |
| 중형 | 15회 | ₩2,500 |
| 대형 | 50회 | ₩6,000 |

**타겟**: 막힌 구간 빠르게 넘기고 싶은 유저

#### 3. 스킨/테마 (선택)
**가격**: ₩1,000~3,000
**내용**:
- UI 색상 테마
- 폰트 변경
- 특수 효과

**타겟**: 개인화 원하는 유저

### IAP 수익 예상

| 지표 | 값 |
|------|-----|
| MAU | 50,000명 (목표) |
| 결제 전환율 | 2~3% |
| 결제 유저 수 | 1,000~1,500명 |
| ARPPU | ₩5,000 |
| 월 수익 | ₩5,000,000~7,500,000 |

## 수익원 3: 확장 콘텐츠 (DLC)

### 에피소드 확장
**구조**: 기본 에피소드 1 무료 → 추가 에피소드 유료

| 에피소드 | 가격 | 내용 |
|----------|------|------|
| EP1: 끝없는 월요일 | 무료 | 기본 스토리 |
| EP2: 화요일의 비밀 | ₩3,000 | 새 스토리라인 |
| EP3: 과거의 루프 | ₩3,000 | 프리퀄 |
| 시즌패스 | ₩5,000 | EP2~3 번들 |

### DLC 수익 예상

| 지표 | 값 |
|------|-----|
| EP1 완료 유저 | 20,000명 |
| EP2 구매율 | 15~20% |
| 구매 유저 | 3,000~4,000명 |
| EP2 가격 | ₩3,000 |
| 수익 | ₩9,000,000~12,000,000 |

## 수익화 타이밍

### 플레이어 여정별 수익화

```
루프 1~3 (튜토리얼)
├── 광고 없음
├── 순수한 게임 경험
└── 리텐션 확보 우선

루프 4~7 (탐색기)
├── 보너스 머니 광고 첫 제안
├── 시간 연장 광고 해금
└── 자연스러운 필요 발생

루프 8~12 (몰입기)
├── 힌트 시스템 활성화
├── 리와인드 광고 해금
├── 프리미엄 패스 구매 유도
└── 주 수익 구간

루프 13+ (마스터)
├── 모든 광고 옵션 활성
├── DLC 구매 유도
└── 히든 콘텐츠 수익화
```

### 수익화 레드라인

❌ 하지 않을 것:
- 강제 광고 (스킵 불가)
- 진행 막는 페이월
- 스탯 구매 (Pay-to-Win)
- 에너지 시스템 (플레이 제한)
- 가챠/랜덤박스
- 미완성 상태 출시 후 유료 완성

## 리스크 및 대응

### 리스크 1: 광고 수익 저조
**원인**: 유저가 광고 필요성 못 느낌
**대응**:
- 광고 보상 가치 상향
- 자연스러운 필요 상황 설계
- A/B 테스트로 최적화

### 리스크 2: 결제 전환 저조
**원인**: 무과금으로 충분히 플레이 가능
**대응**:
- 프리미엄 패스 가치 강화
- 편의성 격차 벌리기 (불편함 증가 X)
- 코스메틱 아이템 추가

### 리스크 3: 리텐션 부족
**원인**: 콘텐츠 소진 후 이탈
**대응**:
- 정기적 콘텐츠 업데이트
- 시즌 이벤트
- 커뮤니티 빌딩

### 리스크 4: 플랫폼 수수료
**현황**: 토스 플랫폼 수수료 정책 확인 필요
**대응**:
- 토스페이 결제 연동
- 플랫폼 특화 프로모션

## 론칭 전략

### 소프트 론칭 (1개월)
- 일부 유저 대상 테스트
- 수익 모델 A/B 테스트
- 버그 및 밸런스 수정

### 정식 론칭
- 마케팅 캠페인
- 론칭 기념 이벤트 (프리미엄 패스 할인)
- 인플루언서 협업

### 포스트 론칭 (3개월)
- 유저 피드백 반영
- EP2 개발 착수
- 시즌 이벤트 진행
