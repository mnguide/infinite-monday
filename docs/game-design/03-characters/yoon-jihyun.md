## 외모

> 밝은 인상, 단정하게 묶은 긴 머리.
> 깔끔한 오피스 캐주얼 차림.
> 웃을 때 보이는 보조개.
> 피곤할 때면 살짝 그늘지는 눈.

### 묘사 예시

```
"엘리베이터 문이 열리자 지현이 서 있었다.
 오늘도 깔끔한 블라우스에 슬랙스 차림.
 눈이 마주치자 환하게 웃는다.
 '오빠, 안녕하세요!'"
```

## 성격

### 표면

- **밝음**: 항상 긍정적이고 활기참
- **사교적**: 누구와도 잘 어울림
- **열정적**: 일에 열심히 임함
- **친절함**: 상대를 배려하는 말투

### 내면

- **불안**: 미래에 대한 걱정
- **외로움**: 겉으로 드러내지 않는 고독
- **야망**: 인정받고 싶은 욕구
- **피로**: 밝은 척 하는 것의 지침

### 성격 변화 (친밀도에 따라)

```
호감도 0~30: 밝고 친절한 동료
호감도 31~50: 장난도 치는 친한 사이
호감도 51~70: 진심을 보여주기 시작
호감도 71+: 솔직한 감정 표현
```
