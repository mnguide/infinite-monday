## 외모

> 성별: 유저 선택 또는 중성적 묘사
>
> 평범한 직장인 외모.
> 특별히 튀지 않는, 어디서나 볼 수 있는 모습.
> 조금 피곤해 보이는 눈, 단정한 복장.

### 묘사 예시

```
"거울에 비친 내 모습은 평소와 다름없다.
 약간 피곤해 보이는 눈, 단정하게 정돈한 머리.
 출근하기 딱 좋은 모습이다. 또 월요일이니까."
```

## 성격

### 기본 성향

- **표면**: 평범하고 무난한 직장인
- **내면**: 현재 상황에 의문을 품기 시작
- **잠재력**: 상황에 따라 적극적으로 변화 가능

### 성격 요소

| 특성 | 정도 | 설명 |
|------|------|------|
| 호기심 | 높음 | 이상한 상황 탐구 욕구 |
| 공감 | 중상 | 동료들에게 관심 |
| 용기 | 가변 | 플레이어 선택에 따라 |
| 도덕성 | 가변 | 플레이어 선택에 따라 |

### 루프에 따른 변화

```
루프 1~3: 혼란, 당황, 불안
루프 4~7: 적응, 탐구심 증가
루프 8~12: 적극적 조사, 목표 의식
루프 13+: 결단, 행동
```

## 배경 스토리

### 과거

- 평범한 대학 졸업
- 취업 준비 끝에 넥스트엣지 입사
- 3년간 성실하게 근무
- 특별한 야망 없이 살아옴

### 현재 (루프 시작 전)

- 매일 비슷한 일상 반복
- 막연한 허무함
- "이대로 괜찮은 건가?" 하는 의문
- 월요일 아침, 이상한 기시감

### 비밀 (스포일러)

> **주의**: 메인 스토리 스포일러 포함

- 입사 시점에 Project Monday 대상자로 선발됨
- "각성 잠재력"이 높은 인물로 분류
- 루프 기억 유지 = 시스템 오류 또는 의도된 설계
- 본인의 과거 선택이 현재 상황과 연결 (후반 공개)

## 게임 내 역할

### 플레이어의 대리인

- 모든 선택은 플레이어가 결정
- 성격/성향은 선택에 의해 형성
- 최소한의 사전 설정만 제공

### 스토리 추진

- 유일하게 루프를 인식하는 존재 (처음엔)
- 조사/추리의 주체
- 관계 형성의 중심

### 감정 이입 대상

- 직장인으로서의 공감 포인트
- 반복되는 월요일의 피로감
- 탈출하고 싶은 욕구

## 스탯 시스템

### 기본 스탯

| 스탯 | 초기값 | 범위 | 역할 |
|------|--------|------|------|
| 체력 (Health) | 100 | 0~100 | 신체적 상태, 0이면 기절 엔딩 |
| 멘탈 (Mental) | 80 | 0~100 | 정신적 상태, 0이면 붕괴 엔딩 |
| 평판 (Reputation) | 50 | 0~100 | 회사 내 이미지, 엔딩 조건 |
| 돈 (Money) | 25000 | 0~ | 사용 가능 자금 |

### 스탯 감소/회복

| 상황 | 체력 | 멘탈 | 평판 |
|------|------|------|------|
| 야근 | -10~20 | -5~15 | +5 |
| 휴식 | +10 | +10 | - |
| 커피 | - | +5 | - |
| 업무 성공 | - | +5 | +10 |
| 업무 실패 | - | -10 | -15 |
| 지각 | -5 | -5 | -10 |

## 플레이어 아크

### 5막 구조

```
1막 (루프 1~3): 발단
   - 상황 인식
   - 혼란과 당황
   - "왜 나만?"

2막 (루프 4~7): 전개
   - 루프 활용법 학습
   - 단서 수집 시작
   - 동료들과 관계 형성

3막 (루프 8~12): 위기
   - 핵심 비밀 접근
   - 딜레마 직면
   - "이대로 괜찮은가?"

4막 (루프 13~15): 절정
   - 진실 대면
   - 중요한 선택
   - 관계의 결실

5막 (루프 15+): 결말
   - 엔딩 분기
   - 플레이어 선택에 따른 결과
```
