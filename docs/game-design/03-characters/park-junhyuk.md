## 외모

> 마른 체형, 창백한 피부.
> 눈 밑에 다크서클이 짙다.
> 항상 후드집업이나 편한 복장.
> 조용히 모니터만 바라보는 모습.

### 묘사 예시

```
"준혁이 모니터에서 눈을 떼지 않는다.
 검은 후드집업, 헤드폰.
 키보드 위로 빠르게 움직이는 손가락.
 무언가에 깊이 빠져있는 듯하다."
```

## 성격

### 표면

- **내성적**: 말이 적고 조용함
- **집중력**: 업무에 깊이 몰두
- **무심함**: 주변에 관심 없어 보임
- **독특함**: 뭔가 다른 느낌

### 내면

- **고독**: 오랜 시간 혼자였음
- **체념**: 포기한 듯한 태도
- **관찰자**: 모든 것을 지켜보고 있음
- **희망**: 아직 완전히 포기하지 않음

### 특별한 비밀

> **핵심 반전**: 준혁도 루프를 인식하고 있다.
> 그것도 플레이어보다 훨씬 오래전부터.
> 현재 **100번째 루프** 중.

### 과거 (2년 전)

- 천재적인 개발 능력으로 주목받음
- 넥스트엣지에 특채 입사
- Project Monday의 "첫 번째 각성자"

### 첫 번째 각성자

```yaml
discovery: |
  준혁은 입사 첫 주에 각성했다.
  루프를 인식한 최초의 피실험자.

  처음엔 탈출하려 발버둥쳤다.
  50번째 루프쯤 포기했다.
```

### 현재

- 플레이어의 각성을 감지
- 처음엔 관망
- 점점 동료 의식 느낌
- 도움을 줄지 고민 중
